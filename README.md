# Tesla Tasker Plugin

Android **Tasker**로 테슬라 차량을 제어하기 위한 Tesla Fleet API 연동용
OAuth 중계(relay) + 가상 키 공개키 호스팅 사이트입니다. (GitHub Pages)

- OAuth 콜백 중계: `callback/index.html`, `callback.html` → `teslatasker://callback`
- 3rd‑party 공개키: `.well-known/appspecific/com.tesla.3p.public-key.pem`
